#Dashboard de Vendas – Excel (XBOX)

## Entendendo o Desafio
O objetivo deste desafio é criar um **dashboard de vendas no Excel**, com foco na **organização, análise e visualização de dados**.  
A proposta é transformar **dados brutos** em **informações visuais claras e úteis**, permitindo uma análise eficaz do desempenho de vendas e apoiando a **tomada de decisões baseada em dados**.

## Objetivo do Projeto
Criar um dashboard de vendas que permita:
- Organizar dados de clientes e planos de forma estruturada
- Analisar o faturamento de planos anuais
- Comparar vendas com e sem renovação automática
- Visualizar resultados de forma clara e intuitiva por meio de gráficos

## Base de Dados Utilizada
Foi utilizada uma base de dados de clientes da **XBOX**, contendo informações como:

- Clientes
- Tipo de plano
- Benefícios incluídos em cada plano
- Indicação se o plano é **automaticamente renovado** ou não
- Indicação se o plano é **anual** ou não
- Valor do plano

Esses dados foram utilizados exclusivamente para fins educacionais e de análise.

## Perguntas de Negócio
O dashboard foi desenvolvido para responder às seguintes perguntas de negócio:

1. **Qual o faturamento total de vendas dos planos anuais**, considerando todas as assinaturas agregadas?
2. **Qual o faturamento total de vendas dos planos anuais**, separado entre:
   - Planos com **renovação automática**
   - Planos **sem renovação automática**

## Tratamento e Análise dos Dados
Para realizar a análise, foram utilizadas:

- **Tabelas Dinâmicas**, responsáveis pelos cálculos e agregações
- Agrupamento dos valores de vendas por:
  - Trimestre
  - Semestre
  - Ano
- Segmentação dos dados com foco na informação:
  - Se o plano é **autorenovado** ou não

Os cálculos foram realizados manualmente a partir das tabelas dinâmicas, garantindo a consistência dos resultados apresentados.

## Dashboard
A visualização final dos dados foi apresentada em um **dashboard no Excel**, com gráficos que destacam:

- Faturamento total dos planos anuais
- Comparativo entre planos com e sem renovação automática
- Evolução das vendas ao longo do tempo (trimestres, semestres e anos)

> Obs: O layout final do dashboard segue o modelo apresentado pelo professor **Felipe**, utilizado como referência visual.
